# <img src="extension/assets/fucksparx.png" style="width: 2rem"> Sparx Hax
Welcome to **Sparx Hax**, a free browser extension which is designed to assist you with your homework through the use of artificial intelligence.

## Disclaimer
This extension uses AI and I can't promise the answers will be correct. Currently AI struggles with certain types of questions, mostly graph questions because it takes what is called a random f***ing guess. If the answer looks messed up, just run the solver again and you will most likely get a different answer. You're just going to have to lock in with graph questions though.

## Installation
If you're reading this I am still waiting on getting the extension submitted to Google Web Store, so you have to clone this repository, open chrome settings from the three dots in the top-right corner and select Extensions then Manage Extensions, or type chrome://extensions/ in the address bar. Then enable developer mode using the toggle at the top-right of the page and click load unpacked at the top left and select the extension folder.

## Setup 
This extension uses Groq AI, so you'll need to obtain an API key from their website https://console.groq.com/keys. Enter this key into the API key tab of the extension and press save.
